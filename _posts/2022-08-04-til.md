1. MongoDB Homebrew Tap 추가

`brew tap mongodb/brew`

2. Homebrew 최신 버전으로 업데이트

`brew update`

3. MongoDB 설치

`brew install mongodb-community@5.0`

4. Service로 MongoDB 실행 및 중지

`brew services start mongodb-community@5.0`
